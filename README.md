# react ssr

=================

## Install

```bash
$ npm install
```

## dev

```bash
$ npm run dev
```

## test

```bash
$ npm run build
$ npm run test
```

## online

```bash
$ npm run build
$ npm start
```

## 项目结构

```
app/controller/*   ------------------服务端controller
app/extent/*       ------------------服务端框架扩展方法
app/public         ------------------服务端静态文件
app/service/*      ------------------服务端service
router.js          ------------------路由文件
client/*           ------------------页面
config/*           ------------------项目配置文件
